# Feature Ideas

## Quiz Delivery Modes

- **Scheduled quizzes** - Send quizzes at configured times automatically
- **Quiz sessions** - Bundle multiple questions (e.g. 10) into a single session, user completes them in one go
- **Single questions** - Send individual standalone questions on demand

## Difficulty Modes

- **Easy** - Multiple choice with 4 answer options
- **Medium** - Multiple choice with fewer options or stricter time limits
- **Hard** - Free typing, user must type the exact answer

## Competitive Features

- **Duel mode** - Two players compete head-to-head answering the same questions, answer time is recorded and compared
- **Challenge a friend** - Send a quiz challenge to a specific user
- **Teams/groups** - Compete as teams, aggregate scores
- **Spectator mode** - Watch duels in real-time

## Progression & Gamification

- **Leaderboards** - Daily/weekly/all-time rankings by score or accuracy
- **Streaks** - Track consecutive days of participation, bonus points for maintaining streaks
- **Achievements/badges** - Unlock for milestones (first perfect session, 100 questions answered, etc.)
- **XP/Levels** - Users gain experience and level up over time

## Question Management

- **Categories/topics** - Tag questions by subject (history, science, etc.), let users filter or focus on weak areas
- **Spaced repetition** - Re-surface questions the user got wrong after some time
- **User-submitted questions** - Let users propose questions for review/approval

## Analytics & Feedback

- **Personal stats** - Accuracy by category, improvement over time, average response time
- **Question analytics** - Track which questions have lowest success rates
- **Explanations** - Show why an answer is correct after answering

## Session Variations

- **Timed rounds** - Complete as many as possible in X minutes
- **Survival mode** - Keep going until you get one wrong
- **Daily challenge** - Same set of questions for everyone each day, compare results
